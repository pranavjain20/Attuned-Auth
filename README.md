# Attuned-Auth

OAuth callback page for [Attuned](https://github.com/pranavjain20/Attuned) — a system that connects WHOOP recovery data to Spotify playlists using neuroscience research.

This page receives OAuth authorization codes from Spotify and WHOOP during user onboarding, displaying them for secure transfer back to the host application. No credentials are stored or transmitted — codes are displayed client-side only.
